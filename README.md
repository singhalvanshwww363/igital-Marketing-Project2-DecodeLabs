## 🧠 Key Concepts Covered

### The IPO Logic Model
`User Query (Demand)` → `Algorithm (Relevance + Authority)` → `The SERP (Result)`

### 91% of Searches are Long-Tail
- **Head keywords** (e.g., "Python") = High competition, Low conversion ❌
- **Long-tail keywords** (e.g., "Remote Data Science Internship for freshers India") = Low competition, High conversion ✅

### KGR Formula
```
KGR = allintitle: results ÷ monthly search volume

< 0.25   → GREAT — Rank in days 🏆
0.25–1.0 → MODERATE — Takes time ⚠️
> 1.0    → BAD — Too competitive ❌
```

### 4 Archetypes of Search Intent
| Intent | Description | Example |
|--------|-------------|---------|
| Informational | Seeking knowledge | "What is machine learning?" |
| Navigational | Looking for a specific page | "DecodeLabs login" |
| Commercial | Comparing options | "Best Python course 2026" |
| Transactional | Ready to act | "Apply data science internship" |
